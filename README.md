# tweets

A simple react application

## Getting Started

Clone the repository in an empty directory.

```
$ git clone 
```

Go to the directory and install the project dependencies.

```
$ npm install
```

Run the server.

```
$ npm run start
```

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
